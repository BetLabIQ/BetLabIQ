# BetLabIQ Fresh Website Starter

This is a clean GitHub Pages-ready starter for BetLabIQ.

## Files
- `index.html` — homepage structure
- `styles.css` — full responsive styling and color palette
- `assets/betlabiq-logo.png` — logo used in the top-left nav
- `assets/whop-banner-reference.png` — optional banner reference asset

## Color Palette
- Deep Navy: `#050A14`
- Navy: `#081220`
- Website Navy: `#0B1220`
- Electric Blue: `#3B82F6`
- Soft Blue: `#60A5FA`
- Cyan: `#06B6D4`
- Bright Cyan: `#22D3EE`
- White: `#F8FAFC`

## Main Gradient
```css
linear-gradient(135deg, #3B82F6 0%, #06B6D4 100%)
```

## Before Going Live
Replace these placeholder links in `index.html`:
- `https://discord.gg/YOUR-FREE-DISCORD`
- `https://discord.gg/YOUR-PREMIUM-DISCORD`

## GitHub Pages Setup
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings → Pages.
4. Deploy from branch: `main` and folder `/root`.
5. Save.
